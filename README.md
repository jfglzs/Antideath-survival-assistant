# 功能列表

## 常用功能

### 设置菜单

* 默认快捷键：`Z + K`
* 打开 ASA 设置菜单。

---

## 生存辅助

### 苦力怕预警器

检测玩家周围一定范围内是否存在苦力怕，并进行提示。

配置：

* 检测范围（默认 8 格）

---

### 剩余物品 Overlay

实时显示：

* 主手剩余数量
* 背包剩余数量
* 潜影盒内剩余数量

支持调整：

* X 偏移
* Y 偏移

---

### 防止被刻意的游戏设计杀死

在部分危险情况下自动进行保护，减少因一些特殊机制导致的死亡。

---

### 低生命值自动执行命令/发送聊天

生命值低于指定值时自动：

* 执行命令
* 或发送聊天消息

可配置：

* 生命值阈值
* 发送内容
* 发送模式

例如：

```
!s
```

---

## 建筑辅助

### 材料待取 Overlay

启用后：

鼠标中键点击原理图中的方块，如果玩家背包中没有对应材料，则自动加入待取列表。

支持：

* Overlay 偏移
* 一键清空待取列表

---

### 中键投影取货（LMS）

需要：

* LMS Carpet Addition

功能：

中键点击投影方块即可自动远程取货。

---

### 右键投影方块取货

同样需要：

* LMS Carpet Addition

右键即可自动请求材料。

---

### 假人自动打开背包

远程取货后：

自动打开假人背包。

支持两种模式：

* 指令模式
* 主手右键模式

---

### 假人自动下线

远程取货完成后自动下线假人。

支持配置：

* 打开背包延迟
* 下线延迟

---

## 假人工具

### 打开假人取货菜单

快捷打开假人取货 GUI。

---

### 假人杀戮光环

快捷键触发。

自动攻击附近符合条件的假人。

支持：

* 检测距离
* 名称前缀
* 白名单
* 黑名单

默认前缀：

```
bot_
```

---

### /pm 批量操作命令

支持批量控制假人。

格式：

```
/pm <前缀> <开始编号> <结束编号> <Action>

/pm <开始编号> <结束编号> <Action>
```

例如：

```
/pm bot_ 1 20 kill
```

支持：

* 默认前缀
* 每次执行后的冷却时间

---

## 自动垃圾清理

自动清理背包中的垃圾物品。

支持：

* 丢弃模式
* 其它模式（根据后续扩展）

支持：

* 白名单
* 黑名单

并提供快捷键：

* 保存当前背包到名单
* 切换清理模式

---

## TAB 菜单增强

### TAB 玩家过滤

过滤：

* 常驻假人
* 无用玩家
* 指定前缀玩家

支持：

* 白名单
* 黑名单
* 前缀过滤

---

### 禁用 TAB 背景

移除 TAB 菜单背景，提高透明度。

---

## MiniHUD

### MiniHUD 掉帧优化

优化 MiniHUD 在部分场景下的性能。

---

### Logger 挂载到 MiniHUD

将 Carpet Logger 信息直接显示到 MiniHUD。

支持：

* 白名单
* 黑名单

匹配方式支持：

```
.contains(xxx)
```

也支持正则表达式（如果启用）。

---

## 聊天增强

### 聊天消息映射

收到聊天内容后自动执行对应命令。

例如：

```
!s=spectator

!h=home
```

发送：

```
!s
```

自动执行：

```
/spectator
```

---

## 客户端优化

### 禁用连接超时

在加载大型原理图等场景下，可避免客户端连接超时。

---

### 禁用加载地形界面

跳过 "Loading Terrain" 界面。

理论上可略微提高进入世界速度。

---

### 随时断开连接

为：

* Connect Screen
* Reconfig Screen

增加退出按钮。

无需等待连接结束即可退出。

---

### MiniHUD FPS 优化

降低 MiniHUD 某些渲染导致的性能损耗。

---

## 渲染优化

提供多个纯客户端渲染优化：

* 禁用玩家盔甲渲染
* 透明展示框
* 禁用掉落物旋转
* 禁用字幕背景
* 禁用容器背景
* 打开材料列表时自动关闭字幕

---

## 挖掘功能（移植自 OMMC）

### 强制方块挖掘冷却

移植自 OMMC。

---

### 平坦挖掘

移植自 OMMC。

---

## 放置保护

### 禁止在地狱门附近放置方块

防止误操作堵门。

支持：

* 白名单

---

# 配置说明

部分功能支持：

* 快捷键
* 白名单
* 黑名单
* 名称前缀
* Overlay 偏移
* 检测范围
* 延迟
* 正则表达式匹配

均可在 ASA 配置界面中进行修改。

---

| 功能           | 依赖                  |
| ------------ | ------------------- |
| 材料待取 Overlay | Litematica          |
| 中键取货         | LMS Carpet Addition |
| 假人远程取货       | LMS Carpet Addition |
| Logger 挂载    | MiniHUD + Carpet    |

# 说明

本模组为纯客户端模组，不修改服务端内容。

大部分功能可自由开启或关闭，仅在对应功能启用时生效。
